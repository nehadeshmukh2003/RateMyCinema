# 🎬 RateMyCinema – Movie Listing and Rating Website

## Project Overview
RateMyCinema is a web-based movie discovery and rating platform that allows users to explore movies, search for specific titles, watch trailers, and provide ratings and feedback after logging into the system. The website uses a real movie database API to fetch movie information dynamically and display it in a clean and responsive user interface.

The main goal of this project is to create an interactive movie browsing platform similar to popular movie websites where users can discover movies, rate them, and share feedback.

## Key Features

### 🎥 Real-Time Movie Data
The website integrates with the TMDB (The Movie Database) API to automatically fetch and display hundreds of movies including movie posters, titles, and ratings.

### ⭐ Movie Rating System
Users can rate movies using a star-based rating system. Ratings are saved locally for each user using browser local storage.

### 🔐 User Authentication
Users can create an account using the signup form and log in to the platform. After login, the navbar displays a logout option and the homepage greets the user with a welcome message.

### 🔍 Movie Search
The search bar allows users to search movies dynamically using the TMDB search API.

### 🎬 Movie Trailer Popup
Users can watch movie trailers directly from the website through a popup window that plays the trailer using YouTube.

### 🌓 Dark / Light Mode
The website supports dark and light themes which can be toggled from the navigation bar for better user experience.

### 💬 Feedback System
Logged-in users can submit feedback about the website and movie experience through a feedback form.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TMDB Movie API
- LocalStorage for user data
- Git & GitHub for version control

## How the System Works

1. The website fetches movie data from the TMDB API.
2. Movies are displayed as responsive movie cards.
3. Users can search movies using the search bar.
4. Users must sign up and log in to rate movies or submit feedback.
5. Ratings are stored in LocalStorage for each movie.
6. Users can watch trailers through a popup player.
